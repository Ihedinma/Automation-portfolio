# Brevo → Airtable Email Engagement Sync

This Make.com scenario captures all email engagement events from Brevo and logs them directly into an Airtable CRM, creating a clean engagement history for every contact.

## Features
- Receives real-time events via Brevo webhook
- Supports opens, clicks, bounces, unsubscribes, deliveries
- Matches the contact in Airtable by email
- Creates a new “Engagement Log” entry for each event
- Updates contact with latest activity info
- Stores full raw event payload for analytics

## Tech Stack
- Make.com
- Brevo Webhooks API
- Airtable Base & API

## Impact
- Centralized engagement analytics
- Accurate contact activity history
- Improved marketing and CRM intelligence
