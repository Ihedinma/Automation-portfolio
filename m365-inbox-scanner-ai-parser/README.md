# Microsoft 365 Inbox Scanner & AI Parser

A Make.com automation that scans a Microsoft 365 inbox, extracts information from incoming emails using AI, and stores the parsed data in a CRM or database.

## Features
- Watches Outlook inbox for new messages  
- Filters by sender/subject/keywords  
- Extracts email body + attachments  
- AI parsing of names, dates, phones, order IDs  
- Normalizes data into structured JSON  
- Saves results to Airtable, Google Sheets, or CRM  

##  Tech Stack
- Make.com  
- Microsoft 365 Outlook API  
- OpenAI GPT-4+  
- Airtable / Sheets (optional)

##  Impact
- Eliminated manual email processing  
- Improved accuracy of extracted data  
- Centralized CRM insights
