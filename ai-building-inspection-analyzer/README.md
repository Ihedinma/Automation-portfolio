# AI Building Inspection Analyzer (Webhook + Multimodal AI)

This Make.com automation receives inspection data via webhook, processes images and notes using multimodal AI, and returns a structured, professional building inspection report.

## Features
- Webhook endpoint for inspection submissions  
- Processes text + images  
- Converts images to base64 for AI analysis  
- Sends multimodal payload to GPT-4 / GPT-4 Vision  
- Generates structured technical assessment  
- Cleans and validates AI output  
- Returns standardized JSON back to client  

## Tech Stack
- Make.com  
- Custom Webhooks  
- OpenAI GPT-4 Vision  
- JSON validation & parsing  

## Impact
- Automated technical reporting  
- Faster inspections  
- Improved accuracy and consistency
